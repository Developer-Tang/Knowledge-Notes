## 实参传递的值与引用

> 方法调用时实参分为值传递和引用传递

> **值传递：**值传递多发生于数值类型参数的传递，以及String类型，修改值传递的变量对其原有的变量不会做成变动
>
> **引用传递：**数组、非基本数据类型为引用传递，修改引用中的数据会对原变量的数据进行改动

## final关键字的作用

> final修饰的类：不可被继承

> final修饰的方法：不可被重写

> final修饰的变量：只能进行一次赋值，赋值后不可以修改指向（内存地址），即基本数据类型不可修改值，引用类型不能改变引用地址，但可以修改引用地址内的值
